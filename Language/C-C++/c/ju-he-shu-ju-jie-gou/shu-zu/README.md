# 数组

在C和C++中，数组是存储相同类型数据的一组连续内存位置的集合。虽然C和C++都支持数组，但它们之间的使用和特性存在一些细微差别。下面是对数组在这两种语言中的简要说明：

#### C中的数组

1.  **声明与初始化**\
    在C语言中，数组的声明语法是：

    ```c
    int arr[5];  // 声明一个可以容纳5个整数的数组
    ```

    初始化数组时可以这样写：

    ```c
    int arr[5] = {1, 2, 3, 4, 5};  // 直接初始化数组
    ```

    如果初始化时元素个数少于数组大小，剩余元素会自动填充为0：

    ```c
    int arr[5] = {1, 2};  // 剩余元素初始化为0
    ```
2.  **数组的访问**\
    数组中的元素可以通过索引访问，索引从0开始。例如：

    ```c
    int firstElement = arr[0];  // 访问数组的第一个元素
    ```
3. **数组的限制**
   * 数组的大小必须是编译时已知的常量，无法动态改变。
   * 无法直接获取数组的大小，必须通过手动维护或传递大小参数。
4.  **指针与数组**\
    在C中，数组名本质上是一个指向数组首元素的指针。例如：

    ```c
    int *ptr = arr;  // 数组名 arr 是一个指向 arr[0] 的指针
    ```

    可以使用指针算术来访问数组元素：

    ```c
    int secondElement = *(arr + 1);  // 等价于 arr[1]
    ```



## 小结：

在C语言中，数组的大小是固定的，无法动态调整。
