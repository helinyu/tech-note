# 声明定义方式



## 1、使用统一的关键字

1.1 不需要指定类型，系统会自动判断

{% code title="JS" %}
```
var a = 12; // 系统会自动推断a的类型是Int类型
```
{% endcode %}



1.2 指不指定类型，都可以

{% code title="swfit" %}
```
var a = 12; // 自动推断
或
var a:int = 12;
```
{% endcode %}



{% hint style="info" %}
常见的声明变量或者常量的关键字有

var

let

const&#x20;

static&#x20;

。。。
{% endhint %}

## 2、使用数据类型

{% code title="C++" %}
```
Int a = 3;
String a = "sd";
// 直接使用类型来强制声明
```
{% endcode %}



{% hint style="info" %}
小结：

从声明变量/常量的方式上都可以看出编程语言的特性。
{% endhint %}
