# Python环境的安装

## 1、直接安装

### Mac电脑上

本来就带有



### Windows上

直接下载安装包安装



## 2、通过conda安装 —— 推荐

先装conda，然后通过conda创建对应的python环境可以设置python的版本



常见的是：

conda create -n xx【名字】 python=版本号

conda activate xx 激活这个环境



在Windows上，通过大概conda的图形界面启动终端。



{% hint style="info" %}
通过conda，我们可以创建不同的python的版本环境来执行不同的项目， —— 推荐
{% endhint %}
