# 标准I/O常量

在C语言的标准I/O库中，有几个常用的常量，主要用于文件操作和流控制。以下是一些重要的标准I/O常量：

#### 文件打开模式常量

* `r`：以只读模式打开文件。如果文件不存在，则打开失败。
* `w`：以写入模式打开文件。如果文件存在，内容会被清空；如果文件不存在，则创建新文件。
* `a`：以追加模式打开文件。数据会被写入到文件末尾。如果文件不存在，则创建新文件。
* `rb`、`wb`、`ab`：以二进制模式打开文件，适用于读取和写入二进制数据。

#### 特殊文件指针常量

* `stdin`：标准输入流，通常与键盘输入关联。
* `stdout`：标准输出流，通常与控制台输出关联。
* `stderr`：标准错误流，用于输出错误信息，通常也与控制台关联。

#### EOF 常量

* `EOF`：表示文件结束的常量，通常为`-1`。在读取文件时，达到文件末尾时会返回此值。

#### BUFSIZ 常量

* `BUFSIZ`：一个定义在`stdio.h`中的常量，表示流缓冲区的默认大小。

#### 其他常量

* `FOPEN_MAX`：可以同时打开的最大文件数，定义在`stdio.h`中。
* `FILENAME_MAX`：单个文件名的最大长度，定义在`stdio.h`中。
