# Task

`Task` 的枚举类型，用于表示HTTP请求的不同任务类型。每个枚举成员代表一种特定的HTTP请求方式或任务类型，包括但不限于：

* 简单请求（无额外数据）
* 带数据的请求
* 带JSON编码数据的请求
* 带自定义JSON编码器的请求
* 带编码参数的请求
* 带组合数据和URL参数的请求
* 文件上传任务
* 多部分表单数据上传任务
* 文件下载任务
