# fossil上的wiki

#### &#x20;**问题跟踪和 Wiki**

*   **查看问题列表**：

    ```bash
    fossil ticket list
    ```

    显示所有问题（tickets）。
*   **添加新问题**：

    ```bash
    fossil ticket create
    ```

    通过命令行或 Web UI 添加新问题。
*   **创建或编辑 Wiki 页面**：

    ```bash
    fossil wiki create page-name
    fossil wiki edit page-name
    ```
